Class 6 - Computer Science Lesson Plan

Book: Avartan Code to AI

Chapter Learning Objectives Teaching Methods Outcomes Activities / Lab Sessions Practical Sessions Estimated Classes
Ch-1: Computer Languages (April 2025) Understand types of computer languages and their evolution. Lecture, presentations, discussions. Students will identify different types of languages and their uses. Group discussion on programming languages. None 3
Ch-2: Formulas and Functions in Excel (April 2025) Learn basic and advanced formulas in Excel. Demonstrations, hands-on practice. Students will apply formulas to perform calculations. Lab session: Creating an Excel sheet with calculations. 1 4
Ch-3: Excel as a Database (July 2025) Understand database concepts using Excel. Interactive demonstrations, real-world examples. Students will use Excel to manage simple databases. Lab session: Creating and managing a student database. 2 4
Ch-4: Exploring GIMP (July 2025) Learn image editing techniques using GIMP. Hands-on training, demonstrations. Students will edit and enhance images. Lab session: Editing an image using various GIMP tools. 2 5
Ch-5: Algorithm and Flowchart (August 2025) Understand problem-solving techniques using algorithms. Step-by-step explanations, problem-solving exercises. Students will create algorithms for basic problems. Activity: Drawing flowcharts for real-life scenarios. None 3
Ch-6: Introduction to HTML5 (October 2025) Learn the basics of web development. Practical demonstrations, coding sessions. Students will create simple web pages. Lab session: Creating a personal homepage using HTML. 3 6
