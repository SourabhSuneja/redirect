Class 7 - Computer Science Lesson Plan

Book: Avartan Code to AI

Chapter Learning Objectives Teaching Methods Expected Outcomes Activities/Labs Practical Sessions Estimated Classes
Ch-1: Number Systems Understanding binary, octal, decimal, and hexadecimal number systems. Lecture, interactive discussions, and problem-solving exercises. Students will be able to convert numbers between different systems. Hands-on worksheet on number conversions. 0 3
Ch-2: Charts in MS Excel Creating and analyzing different types of charts in Excel. Demonstration and hands-on practice in the computer lab. Students will create and interpret bar, pie, and line charts. Lab session on creating a data-driven chart. 1 3
Ch-3: More on GIMP Learning advanced image editing techniques using GIMP. Live demonstrations and practice exercises. Students will enhance and manipulate images effectively. Creating a poster using layers and effects. 2 4
Ch-4: Lists and Tables in HTML5 Understanding the use of lists and tables in web design. Code walkthroughs and live HTML coding. Students will be able to structure content using lists and tables. Creating a basic webpage with lists and tables. 2 4
Ch-5: Images, Links and Frames in HTML5 Using images, hyperlinks, and frames in HTML webpages. Interactive tutorials and hands-on coding. Students will design a webpage with images and links. Creating a mini website with navigation. 2 5
Ch-6: Introduction to Python Understanding basic Python syntax and operations. Hands-on coding sessions and exercises. Students will write simple Python programs. Writing basic Python scripts. 3 6
Ch-7: More on Python Exploring loops, conditions, and functions in Python. Interactive coding activities and problem-solving tasks. Students will write Python programs with loops and conditions. Building a simple calculator using Python. 3 6
Ch-8: Robotics Understanding the basics of robotics and automation. Videos, discussions, and hands-on robot programming. Students will understand how robots function. Simple robot simulation using Scratch. 2 5
Ch-9: Digital Footprint and Cyber Safety Learning online safety and responsible digital behavior. Case studies and role-playing activities. Students will understand how to stay safe online. Group discussion on social media ethics. 0 3
