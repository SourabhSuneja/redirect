Class 11 - Informatics Practices (IP) Lesson Plan - April 2025

Book: CBSE Informatics Practices

Week Chapter Topics Covered Teaching Methods Outcomes Activities / Lab Sessions Estimated Classes
Week 1 Unit-1: Introduction to Computer System Evolution of Computing, Computer Components Lecture, Presentations, Discussions Students will understand the history and architecture of computers. Discussion on past and modern computing trends. 5
Week 2 Unit-1: Introduction to Computer System Types of Software, Memory, Data Security Lecture, Case Studies, Real-world Examples Students will classify software types and understand data security. Case study on cybersecurity threats. 4
Week 3 Unit-2: Introduction to Python Introduction to Programming and Python, Basics of Python (Variables, Syntax etc) Interactive Demonstrations, Hands-on Practice Students will understand the fundamentals of Python programming. Writing and executing basic Python programs. 4
Week 4 Unit-2: Introduction to Python Rules for Naming Variables, Creating Simple Programs Live Coding, Practice Exercises Students will write structured Python programs. Hands-on session on variable usage and Python syntax. 4
