Class 11 - Informatics Practices Lesson Plan

Book: CBSE Informatics Practices Textbook

Unit Topics Learning Objectives Teaching Methods Activities & Practical Sessions Estimated Classes
Unit 1
Introduction to Computer System		 Evolution of computing, components, memory, data security. Understand computing devices, hardware, software, and security. Lecture, discussion, demonstration. Lab: Identifying hardware components. 6 classes
Unit 2
Introduction to Python		 Python basics, control structures, lists, dictionaries, NumPy. Develop problem-solving skills using Python programming. Hands-on coding, interactive exercises. Lab: Writing Python scripts, debugging programs. 14 classes
Unit 3
Database & SQL		 Database concepts, relational model, MySQL, SQL queries. Understand database management, query optimization. Practical SQL queries, database creation. Lab: Executing SQL queries in MySQL. 12 classes
Unit 4
Emerging Trends		 AI, ML, IoT, Cloud Computing, Blockchain. Analyze emerging trends and their impact. Case studies, videos, discussions. Presentation on emerging technologies. 8 classes
