Class 7 - Computer Science

Topic Coverage Plan (April 3 - April 5, 2025)

Date Topic Learning Objectives Resources Used Pedagogy Learning Outcomes Art Integrated Activity Assignment
April 3, 2025 Understanding Different Types of Number Systems Explain binary, decimal, octal, and hexadecimal number systems. Textbook, Digital Slides, Videos Lecture, Visual Aids Students will differentiate between number systems. Draw a chart comparing number systems. Write definitions and examples.
April 4, 2025 Learning Conversion between Number Systems Teach conversion techniques (Binary to Decimal, etc.). Worksheet, Online Converter Hands-on practice, Group Activity Students will successfully convert numbers between systems. Create a visual flowchart for conversions. Solve 5 number conversions.
April 5, 2025 Practice and Application of Number System Conversions Reinforce conversion techniques through exercises. Practice Questions, Class Discussion Interactive Q&A, Problem-solving Students will apply conversion skills in real scenarios. Prepare a presentation on number system applications. Complete book exercises.
